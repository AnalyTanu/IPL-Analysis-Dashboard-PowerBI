# IPL Analysis Dashboard (2008–2025)

A comprehensive Power BI dashboard analyzing 17 seasons of IPL data — from 2008 to 2025.

## Dashboard Preview
![IPL Dashboard](Screenshot%202026-04-10%20140409.png)

## Overview
This dashboard provides end-to-end analysis of IPL matches, players, and team performance across all seasons.

## Key Insights
- Season-wise Champion and Runner-up tracking
- Orange Cap & Purple Cap winners per season
- Total 4s, 6s, Centuries, Half Centuries analysis
- Points Table with team standings
- Player-wise performance — runs, wickets, fours, sixes

## Tools Used
- Power BI (DAX, Data Modeling, Power Query)
- Dataset: Kaggle — IPL Ball by Ball & Match Data

## Dataset
Due to GitHub file size limits, full dataset available on Kaggle:
- Ball by Ball Data
- Match Data (2008–2025)
- Players Data
- Teams Data

## Pages
- Season Overview — Champion, Runner Up, KPIs
- Orange Cap & Purple Cap Stats
- Points Table
- Player Performance Cards
